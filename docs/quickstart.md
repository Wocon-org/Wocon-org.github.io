# Quick Start

## First Launch

When you first open Wocon, you'll be greeted with an interactive world map. Here's how to get started:

### 1. Explore the Interface

- **Top Bar**: Contains the Wocon logo, map layer controls, and user actions
- **Sidebar**: Provides access to core features (Home, Connections, Search, Discover, Plugins)
- **Map Area**: The main interactive map powered by Leaflet.js
- **Control Buttons**: Located at the bottom right (zoom controls) and bottom center (location button)

### 2. Find Your Location

1. Click the location button (üìç) at the bottom right of the map
2. Allow Wocon to access your location
3. The map will automatically zoom to your current position
4. A blue-green marker will appear indicating your location

### 3. Discover New Destinations

1. Click the "Discover" icon in the sidebar
2. Wocon will randomly select a popular travel destination
3. Watch as the map smoothly transitions to the new location
4. A notification will appear showing the destination name
5. The notification will automatically fade after 1.5 seconds

### 4. Search for Places

1. Click the "Search" icon in the sidebar
2. Enter the name of a city or landmark in the search bar
3. Select a result from the dropdown menu
4. The map will zoom to the selected location

### 5. Manage Trips

1. Click the "Home" icon in the sidebar
2. View your existing trips or create a new one
3. Add destinations to your trip by clicking on the map
4. Share your trip with friends or find travel companions

## Basic Navigation

### Map Controls

- **Zoom In/Out**: Use the + and - buttons at the bottom right
- **Pan**: Click and drag the map
- **Change Layer**: Use the layer control in the top right to switch between map styles
- **Reset View**: Double-click anywhere on the map to reset zoom

### Sidebar Navigation

- **Home**: Access your trips and main dashboard
- **Connections**: View and manage your travel connections
- **Search**: Find specific locations
- **Discover**: Randomly explore new destinations
- **Plugins**: Access additional features and integrations

## Tips for New Users

1. **Start with Discover**: Get familiar with the map interface by using the Discover feature
2. **Try Location Services**: Test your location accuracy in different areas
3. **Explore Map Layers**: Switch between different map styles to find your preference
4. **Check Notifications**: Keep an eye on the notification area for updates
5. **Experiment with Zoom**: Practice zooming in and out to understand map scales
